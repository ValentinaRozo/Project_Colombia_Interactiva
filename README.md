# Project_Colombia_Interactiva
Colombia Interactiva is an app for Hololens 2 in wich the user can get to know Colombia Fauna and Flora per department. This Demo contains the main information (Department and Temperature) for Vaupés, Vichada, Valle del Cauca, Atlántico, Antioquia, Amazonas, Arauca and Magdalena. The Demo also contains the interaction of the Fauna and Flora of Amazonas and Magdalena.
